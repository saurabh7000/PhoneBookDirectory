# Steps to run the application

 1.Open the terminal.
 2.Navigate to the correct directory.
 3.Run the command npm init, then change the entry point to "server.js" and confirm other prompts.
 4.Install all dependencies.
 5.Open a terminal and execute one of the following commands: node server.js, nodemon server.js, or npm run server.


# Depedencies
 express
 dotenv
 bcrypt
 cookie-parser
 jsonwebtoken
 mysql2
 validator
 nodemon

# How to install dependencies
 1.Open the terminal.
 2.Navigate to the correct directory.
 3.Run either npm i "dependency-name" or npm install "dependency-name".

# NOTE
  Ensure that in the package.json file, "type": "module" is added.
